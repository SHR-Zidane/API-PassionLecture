# Passion Lecture - Backend API

## Description
Développement du backend d'une application de partage de lectures réalisée dans le cadre du module **P_WEB295** à l'**ETML**.

## Technologies utilisées
* **Runtime :** Node.js & npm 
* **Framework :** Adonis.JS 
* **Base de données :** MySQL
* **Documentation API :** Swagger 
* **Tests :** Bruno 

## Installation et Lancement
1. Cloner le dépôt.
2. Installer les dépendances : `npm install`
3. Configurer le fichier `.env` (Base de données, Clés d'application).
4. Lancer les migrations : `node ace migration:run`
5. Lancer le serveur : `node ace serve --watch`

## Fonctionnalités principales
* API REST complète (Livres, Catégories, Utilisateurs).
* Authentification via jetons OAT.
* Validation rigoureuse des données.
